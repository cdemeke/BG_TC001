SugarClock

A glanceable CGM display for desks, nightstands, and shared family spaces.

SugarClock turns a low-cost Ulanzi TC001 smart pixel clock into an open-source display for people with diabetes and families who want glucose numbers visible without opening a phone.

  • No coding required
  • Runs locally
  • Works with Nightscout / Dexcom Share
  • No subscription
Install Now View GitHub

Built on about $40 hardware, open source, and designed as a convenience display for your home.

Why build it yourself?

Most CGM displays are expensive, locked down, or hard to customize. SugarClock is different: it is affordable, open, and made for everyday visibility.

Install

You're a few minutes away from a working display.

Use Chrome or Edge on a computer. SugarClock guides you through each step, and the Mac setup app is available when browser installation is not supported.

Install with the Mac App

Your browser does not support direct USB installation, so the app walks you through the same setup.

  1. Download and open the SugarClock Setup app on your Mac.
  2. Plug in the clock with a USB-C data cable.
  3. Follow the prompts to install SugarClock and connect Wi-Fi.

Install SugarClock

Browser-based setup for the Ulanzi TC001 smart pixel clock.

These steps assume you already have a Ulanzi TC001 smart pixel clock. Need one first? Use the buying links above.

  1. Plug in the clock with a USB-C data cable.
  2. Click Install and choose the clock when your browser asks.
  3. Finish setup locally by following the prompt on the display.
You may hear beeping during installation. That is expected while the clock updates, and it stops when setup is complete.

SugarClock is not a medical device. Use your CGM receiver, app, or meter for medical decisions.
